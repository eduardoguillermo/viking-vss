💾 Backup y restauración de datos
Exportar datos

Descarga todos los datos en un archivo JSON. Guardalo en tu carpeta de OneDrive para que se sincronice automáticamente con la nube.

Importar / restaurar

Carga un archivo de backup previamente exportado. Atención: reemplaza todos los datos actuales.

Borrar todos los datos

Elimina todos los clientes, dispositivos, historial y configuración. Esta acción no se puede deshacer. Recomendamos exportar un backup antes.

📲 Instalación como PWA

Para instalar esta app en tu dispositivo como si fuera una aplicación nativa, el sistema debe estar publicado en un servidor web (no como archivo local). Opciones recomendadas:

GitHub Pages
Gratis. Subís el archivo a GitHub y queda publicado en una URL fija.
Netlify
Gratis. Arrastrás la carpeta al sitio y obtenés URL en segundos.
Servidor local
XAMPP o Node.js en tu PC. La URL sería localhost.

Los archivos manifest.json y sw.js ya están preparados en el código. Solo necesitás publicar.
